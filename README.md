# Credit Card Fraud Exploratory Analysis

## Description
A short exploratory data analysis of credit card transactions to detect and understand fraudulent activity. Includes data cleaning, visualization, and initial observations.

## Files
- `cleaned_dfزparquet` — cleaned dataset used for analysis
- `credit_card_transactions.csv` — original transactions dataset
- `code.ipynb` — Jupyter notebook with the EDA and visualizations
- `Home.py` — auxiliary script

## How to run
1. Open `code.ipynb` in Jupyter Notebook or JupyterLab.
2. (Optional) Activate the virtual environment in `eda_env` before running the notebook.

## Notes
- Created for exploratory analysis and demonstration. Not intended as a production fraud-detection system.
